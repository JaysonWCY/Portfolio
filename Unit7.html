Object-oriented programming follows the main principles of encapsulation, polymorphism, and inheritance. These main principles are used to reach the goal of having organised, easily understandable, reusable, scalable code.

The use of data structures helps provide a simple framework to manipulate and organise data. An example would be dictionaries. This allows the assigning of a value or attribute to an object, which can be used for data manipulation and organisation.

Lists, whether stacks or queues, allow for the consideration of the order to be of value. As lists can have a variable size, this allows for scalability. Lists also provide the possibility of iteration as the values can easily be changed.

The usage of trees can help with polymorphism as well as inheritance. As the root node branches into children’s nodes, the children’s nodes will inherit attributes from the parent node. Polymorphism can be closely used with tree data structures as the nodes will have similar attributes.
